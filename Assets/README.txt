Extra Features:
- Procedural terrain generation
- Regenerate random terrain during run time (Press T)
- Mouse controls view angle (Click and drag) and distance (scroll)
- Camera follows object automatically

Mouse Controls:
+--------------------+---------------------------------------------------+
| Action             | Effect                                            |
+--------------------+---------------------------------------------------+
| Drag Horizontally  | Camera rotate left and right around object        |
+--------------------+---------------------------------------------------+
| Drag Vertically    | Camera rotate up and down around object           |
+--------------------+---------------------------------------------------+
| Scroll Up and Down | Change camera distance to the object              |
+--------------------+---------------------------------------------------+
